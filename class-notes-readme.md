# Class Notes App

A simple and efficient application for organizing and storing your class notes digitally.

## Features

- **Create Notes**: Quickly jot down notes during lectures or study sessions
- **Organize by Subject**: Categorize notes by class or subject for easy retrieval
- **Search Functionality**: Find specific notes using keywords or topics
- **Edit and Update**: Modify your notes anytime to add more information
- **Date Tracking**: Automatically timestamps notes for reference
- **Offline Access**: Access your notes even without internet connection

## Usage

1. **Adding a Note**: Click the "New Note" button and select your subject
2. **Editing a Note**: Click on any existing note to open and edit it
3. **Searching**: Use the search bar at the top to find notes by keyword
4. **Deleting**: Select a note and click the delete icon to remove it
